Paste some code above and press Build the tour. codetour will read the structure and propose an ordered set of stops—entry points first—that you can reorder, annotate, and export. Or press Load a sample to see it work.

Heuristic, not semantic. codetour finds landmarks with pattern-matching—it does not run or truly understand your code, and it does not use AI. It can miss or mislabel declarations, especially in unusual formatting or unsupported languages. The explanations are yours to write; treat the auto-detected structure as a starting point, not truth.
